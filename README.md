# AZURE - SENTINEL
**En esta practica aprenderas a crear el recurso de Microsoft Sentinel**

## Requisitos
- Tener una suscripcion en Azure
- Tener una conexion a internet
- Tener un navegador instalado (Brave,Google,Firefox, etc)
-------------------------
